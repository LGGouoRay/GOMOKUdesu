# Gomoku 房間列表和大廳功能 - 快速開始指南

## 新增功能概覽

這次更新為 Gomoku 遊戲添加了完整的房間列表和遊戲大廳系統。

## 快速使用步驟

### 方式 1: 本地測試（同一台電腦）

#### 步驟 1: 運行遊戲 - Host 版本
1. 啟動遊戲
2. 進入 **主菜單** → **多人遊戲**
3. 選擇 **Host Game**
4. 進入 **遊戲大廳**，你會看到：
   - "Player 1 (Host/BLACK)" - 綠色指示燈亮起
   - "Waiting for Player 2..." - 灰色指示燈
   - 顯示你的 IP 地址和連接端口（Port: 55001）

#### 步驟 2: 運行遊戲 - Client 版本
1. 在同一台電腦上再啟動一個遊戲實例
2. 進入 **主菜單** → **多人遊戲**
3. 選擇 **Connect Manually**
4. 會自動使用 IP: 127.0.0.1 連接
5. 連接成功後進入 **遊戲大廳**

#### 步驟 3: 開始遊戲
1. 在 Host 的大廳中，現在應該看到：
   - "Player 1 (Host/BLACK)" - 綠色
   - "Player 2 (Client/WHITE)" - 綠色
2. **只有 Host** 可以點擊 "Start Game" 按鈕
3. 點擊 "Start Game" 開始遊戲
4. 遊戲開始，Host 執黑棋，Client 執白棋

### 方式 2: 區域網路測試（多台電腦）

#### 前置準備
- 確保兩台電腦連接到同一個區域網路
- 查詢 Host 電腦的 IP 地址

#### 查詢 Host 的 IP 地址
**Windows:**
```cmd
ipconfig
```
尋找 "IPv4 Address"，例如：192.168.1.100

**Mac/Linux:**
```bash
ifconfig
```
尋找 "inet" 地址

#### 步驟
1. Host 電腦：進入 **多人遊戲** → **Host Game**
2. Client 電腦：進入 **多人遊戲** → **Connect Manually**
3. Client 輸入 Host 的 IP 地址（例如：192.168.1.100）
4. 連接成功後進入大廳
5. Host 點擊 "Start Game" 開始遊戲

## 菜單選項說明

### 多人遊戲菜單

1. **Host Game**
   - 創建一個房間並等待玩家加入
   - 進入遊戲大廳
   - 在大廳中看到你的 IP 地址
   - 當有玩家加入時按 "Start Game" 開始遊戲

2. **Find Room**
   - 搜索區域網路中可用的房間
   - 列出發現的房間
   - 選擇房間加入
   - 進入遊戲大廳
   - **注意**: 目前 UDP 廣播還在開發中，此功能框架已準備

3. **Connect Manually**
   - 手動輸入 Host 的 IP 地址
   - 本地測試預設為 127.0.0.1
   - 區域網路測試輸入實際 IP 地址
   - 連接成功後進入遊戲大廳

## 遊戲流程

### Host 的遊戲流程
```
進入主菜單
    ↓
選擇「多人遊戲」
    ↓
選擇「Host Game」
    ↓
進入遊戲大廳
    ├─ 等待 Client 連接
    ├─ Client 連接後「Start Game」變可用
    └─ 點擊「Start Game」開始遊戲
    ↓
進行遊戲（Host 執黑棋）
    ↓
遊戲結束
```

### Client 的遊戲流程
```
進入主菜單
    ↓
選擇「多人遊戲」
    ↓
選擇「Connect Manually」（或「Find Room」）
    ↓
連接到 Host
    ↓
進入遊戲大廳
    └─ 等待 Host 點擊「Start Game」
    ↓
進行遊戲（Client 執白棋）
    ↓
遊戲結束
```

## 常見問題

### Q: 為什麼 "Start Game" 按鈕是灰色的？
**A:** 這是正常的！只有當另一個玩家連接後，按鈕才會變為可用狀態。在大廳中查看 "Player 2" 是否已連接。

### Q: Client 連接失敗了怎麼辦？
**A:** 
- 確認 IP 地址正確（查看 Host 大廳中顯示的 IP）
- 確認兩台電腦在同一個網路
- 檢查防火牆設置
- 嘗試重新連接

### Q: 可以在不同的網路（如互聯網）中連接嗎？
**A:** 目前只支持區域網路。如果需要跨互聯網連接，需要設定端口轉發或使用 VPN。

### Q: 大廳中可以聊天嗎？
**A:** 目前不支持。這是一個可以在未來添加的功能。

## 技術細節

### 網路架構
- **TCP** 用於遊戲數據傳輸
- **UDP** 預留用於房間廣播發現（待實現）
- **主機端口**: 55001
- **廣播端口**: 55002

### 玩家分配
- **Host 玩家**: 執黑棋（Color: BLACK）
- **Client 玩家**: 執白棋（Color: WHITE）

## 支持的連接方式

? 本地連接 (127.0.0.1)  
? 區域網路 (192.168.x.x, 10.0.x.x, etc.)  
? 廣播房間發現（UDP 廣播，開發中）  
? 互聯網連接（需要額外配置）

## 故障排除

如果遇到問題，嘗試：

1. **重新編譯**
   ```bash
   cmake --build . --clean-first
   ```

2. **檢查連接**
   - 確認防火牆沒有阻止 55001 端口
   - 確認兩台電腦在同一個網路
   - 嘗試 ping Host 的 IP 地址

3. **查看日誌**
   - 遊戲會在控制台輸出網路連接信息
   - "[Network]" 前綴的消息是網路相關
